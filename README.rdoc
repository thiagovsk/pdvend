- Para rodar a aplicação
  - $ git clone 
  - $ bundle install
  - $ rake db:create && rake db:migrate
  - $ rails server

- Para rodar os testes
  - $ rake db:test:prepare
  - $ bundle exec rspec (para rodar os testes)
